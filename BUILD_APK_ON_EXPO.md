# Build APK Using Expo Dashboard (Browser)

## Step 1: Login to Expo
1. Go to: **https://expo.dev/login**
2. Login with your account (nks-16)

## Step 2: Go to Your Project
1. Visit: **https://expo.dev/accounts/nks-16/projects/mediguardian**
2. Click on **"Builds"** tab

## Step 3: Create New Build
1. Click **"Create a build"** or **"New build"** button
2. Select:
   - **Platform:** Android
   - **Build profile:** preview (or production if you want)
   - **Build type:** APK
   
3. Click **"Build"**

## Step 4: Wait for Build
- Build takes 2-5 minutes
- You'll see build progress in real-time
- Status will show: Queued → In Progress → Finished

## Step 5: Download APK
1. Once finished, click **"Download"** button
2. Or scan the QR code to install directly on your phone
3. Install on your device

---

## Alternative: Use EAS CLI in Browser Terminal

If the above doesn't work, you can also:

1. Go to: **https://expo.dev/accounts/nks-16/projects/mediguardian/builds**
2. Click **"Create Build"**
3. Follow the wizard
